¿Qué es Spectra?

WiFi como sensor de percepción

Las señales WiFi llenan cada rincón de un espacio. Cuando una persona se mueve, su cuerpo perturba esas señales. Spectra analiza esas perturbaciones en tiempo real para detectar, localizar y clasificar presencia humana.

📡

Señales WiFi (CSI)

Extraemos Channel State Information de 64 subportadoras OFDM entre transmisores ESP32. Cada frame contiene amplitud y fase de cada subportadora.

🧠

Algoritmos en Tiempo Real

Procesamiento por varianza, análisis de zonas de Fresnel y correlación de subportadoras. Detección en menos de 100ms con baja latencia.

🎯

Detección Precisa

Presencia, movimiento, respiración, velocidad y posición 2D por triangulación con múltiples enlaces WiFi.

Cómo funciona

De ondas WiFi a percepción espacial

1

Transmisión

Las señales WiFi de los ESP32 llenan el espacio con ondas electromagnéticas a 2.4GHz.

2

Perturbación

El cuerpo humano absorbe y refleja las ondas, alterando la amplitud y fase de las subportadoras.

3

Análisis

Procesamos en tiempo real la varianza, zonas de Fresnel y correlación entre subportadoras.

4

Detección

Determinamos presencia, posición, movimiento, velocidad y actividad con alta precisión.

Privacidad y Ética

Tecnología responsable

Spectra está diseñado con la privacidad como principio fundamental. No es una herramienta de espionaje — es una plataforma de percepción para salud, rescate y automatización.

✓

Procesamiento local
Todo se procesa en tu Raspberry Pi. Ningún dato sale de tu red.

✓

Sin cámaras
No captura imágenes ni video. Solo analiza perturbaciones de señal WiFi.

✓

Open source
Código 100% abierto y auditable. Transparencia total sobre qué hace y cómo.

✓

Sin nube
No requiere internet ni servicios externos. Funciona completamente offline.

✓

Consentimiento
Diseñado para uso con consentimiento explícito de los ocupantes.

✓

Anonimización
No identifica personas individuales. Solo detecta presencia y movimiento.

Roadmap

Evolución del proyecto

Fase 1 — Completada

Detección de Presencia

Análisis de varianza de subportadoras para detectar presencia/ausencia con 95% de precisión.

Fase 2 — En progreso

Triangulación 2D

4 nodos ESP32 en configuración cruzada para estimar posición X,Y en la habitación.

Fase 3 — Planificada

IA + Clasificación de Actividad

Redes neuronales para clasificar actividades: caminar, sentarse, dormir, respirar.

Fase 4 — Investigación

Reconstrucción de Pose Humana

Modelos transformer para reconstruir esqueleto humano a partir de señales RF, como WiFi-Kinect.
